# 干系人管理

## 干系人管理过程

### 1. 识别干系人
- 识别受项目影响或影响项目的个人/组织
- 分析其利益、影响力、态度

### 2. 规划干系人参与
- 制定干系人参与策略
- 编制干系人管理计划

### 3. 管理干系人参与
- 沟通协调
- 引导干系人支持

### 4. 监督干系人参与
- 评估参与程度
- 调整策略

## 干系人分类模型

### 权力/利益方格

|          | 高利益 | 低利益 |
|----------|--------|--------|
| **高权力** | 重点管理 (Manage Closely) | 随时告知 (Keep Informed) |
| **低权力** | 随时告知 (Keep Informed) | 监督 (Monitor) |

### 凸显模型

| 类型 | 特征 |
|------|------|
| **权力** | 组织内地位高 |
| **合法性** | 正式要求参与 |
| **紧迫性** | 需立即关注 |

> 三者兼具 = 关键干系人

### 影响方向

| 方向 | 说明 |
|------|------|
| **向上** | 高层领导 |
| **向下** | 团队成员 |
| **向外** | 外部干系人 |
| **横向** | 同级部门 |

## 干系人登记册

| 内容 | 说明 |
|------|------|
| 基本信息 | 姓名、职位、部门 |
| 分类 | 内部/外部、支持/反对 |
| 需求 | 信息需求、期望 |
| 影响力 | 权力、态度 |
| 策略 | 参与方式 |

## 干系人分析

### 分析步骤

1. 识别所有干系人
2. 分析利益需求
3. 评估影响力和态度
4. 分类并制定策略

### 分析方法

- 干系人映射
- 专家判断
- 数据收集（问卷、访谈）
- 数据分析

## 干系人参与策略

| 策略 | 适用对象 |
|------|----------|
| 密集沟通 | 关键干系人 |
| 定期汇报 | 一般干系人 |
| 重点跟进 | 反对者 |
| 主动引导 | 中立者 |
