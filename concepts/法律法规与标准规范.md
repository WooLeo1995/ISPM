# 法律法规与标准规范

## 知识产权

### 著作权 (Copyright)

| 要点 | 说明 |
|------|------|
| 保护对象 | 文学、艺术、科学作品 |
| 保护内容 | 复制权、发行权、改编权等 |
| 保护期限 | 作者终生+死后50年 |
| 取得方式 | 自动取得 |

### 专利权 (Patent)

| 要点 | 说明 |
|------|------|
| 保护对象 | 发明、实用新型、外观设计 |
| 保护内容 | 独占实施权 |
| 保护期限 | 发明20年、实用新型10年 |
| 取得方式 | 申请审批制 |

### 商标权 (Trademark)

| 要点 | 说明 |
|------|------|
| 保护对象 | 商品/服务标志 |
| 保护内容 | 专用权 |
| 保护期限 | 10年，可续展 |
| 取得方式 | 注册取得 |

## 合同法

### 合同类型

| 类型 | 说明 |
|------|------|
| **要式合同** | 须书面形式 |
| **不要式合同** | 口头即可 |

### 合同无效

1. 欺诈/胁迫
2. 恶意串通
3. 合法形式掩盖非法目的
4. 违反强制性规定

### 违约责任

| 形式 | 说明 |
|------|------|
| 继续履行 | 强制履行 |
| 补救措施 | 修复、更换 |
| 赔偿损失 | 支付违约金 |
| 定金 | 双倍返还 |

## 招标投标法

### 招标方式

| 方式 | 说明 |
|------|------|
| **公开招标** | 面向社会 |
| **邀请招标** | 特定供应商 |

### 禁止行为

1. 围标、串标
2. 挂靠投标
3. 低于成本报价
4. 黑白合同

## 政府采购法

| 方式 | 说明 |
|------|------|
| 公开招标 | 主要方式 |
| 竞争性谈判 | 竞争不充分 |
| 单一来源 | 唯一供应商 |
| 询价 | 标准化产品 |

| **计算机软件配置管理计划 (GB/T 12505)** | SCM计划规范 |

### 信息安全标准

| 标准 | 说明 |
|------|------|

| **ISO 27001** | 信息安全管理体系 |
