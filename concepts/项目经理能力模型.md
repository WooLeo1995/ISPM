# 项目经理能力模型

## PMI 人才三角

| 能力 | 说明 |
|------|------|
| **技术项目管理** | 专业知识、方法论 |
| **领导力** | 引导、激励团队 |
| **战略与商业管理** | 业务理解、战略思维 |

## 技术项目管理

| 技能 | 说明 |
|------|------|
| 计划编制 | 制定可行计划 |
| 进度控制 | 确保按时交付 |
| 成本控制 | 控制在预算内 |
| 质量管理 | 确保质量达标 |
| 风险管理 | 识别和应对风险 |
| 沟通管理 | 有效信息传递 |

## 领导力技能

### 领导风格

| 风格 | 特点 | 适用场景 |
|------|------|----------|
| **指令型** | 直接指挥 | 紧急情况 |
| **教练型** | 培养发展 | 能力提升 |
| **参与型** | 共同决策 | 需要共识 |
| **授权型** | 充分信任 | 成熟团队 |

### 情商 (EQ)

| 维度 | 说明 |
|------|------|
| 自我认知 | 了解自己 |
| 自我管理 | 控制情绪 |
| 社会认知 | 理解他人 |
| 关系管理 | 维护关系 |

## 战略与商业思维

| 能力 | 说明 |
|------|------|
| 业务分析 | 理解业务需求 |
| 行业知识 | 掌握行业动态 |
| 竞争分析 | 了解竞争对手 |
| 价值创造 | 为组织创造价值 |

## 项目经理角色

| 角色 | 职责 |
|------|------|
| **发起人** | 争取资源、支持 |
| **沟通者** | 信息传递 |
| **协调者** | 跨部门协调 |
| **决策者** | 关键决策 |
| **问题解决者** | 处理异常 |

## 能力发展路径

```
初学者 → 经验者 → 熟练者 → 专家 → 大师
  ↓        ↓        ↓       ↓      ↓
执行任务  管理项目  领导项目  变革推动  战略贡献
```

## 常见项目经理类型

| 类型 | 特点 |
|------|------|
| **项目推动者** | 关注过程和计划 |
| **团队协调者** | 关注人和关系 |
| **问题解决者** | 关注技术和方案 |
| **变革领导者** | 推动组织变革 |

## 认证体系
