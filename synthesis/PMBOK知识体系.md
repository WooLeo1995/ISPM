# PMBOK知识体系总结

*基于PMBOK第6/7版，项目管理十大知识域与五大过程组*

---

## 一、五大过程组

| 过程组 | 英文 | 说明 |
|--------|------|------|
| **启动过程组** | Initiating | 确定项目愿景、目标，获得授权 |
| **规划过程组** | Planning | 建立范围、制定计划，确定行动方案 |
| **执行过程组** | Executing | 整合资源，完成可交付成果 |
| **监控过程组** | Monitoring & Controlling | 跟踪、审查、调整进度与绩效 |
| **收尾过程组** | Closing | 正式验收、总结经验教训 |

---

## 二、十大知识域与49个过程

### 十大知识域说明

#### 2.1.1 项目整合管理
**核心：** 协调所有知识域，确保项目各要素协调一致

**关注点：**
- 项目与组织战略对齐
- 方案制定和收益实现
- 整合所有子计划
- 变更统一控制

#### 2.1.2 项目范围管理
**核心：** 确保做且仅做所需工作

**关注点：**
- 明确项目边界
- 防止范围蔓延
- WBS分解结构
- 需求可追踪

#### 2.1.3 项目进度管理
**核心：** 确保按时交付

**关注点：**
- 活动定义与排序
- 工期估算
- 进度控制
- 关键路径管理

#### 2.1.4 项目成本管理
**核心：** 确保在预算内完成项目

**关注点：**
- 成本估算
- 预算制定
- 成本控制
- 挣值分析

#### 2.1.5 项目质量管理
**核心：** 满足项目需求和标准

**关注点：**
- 质量标准确定
- 质量保证活动
- 质量控制验证
- 持续改进

#### 2.1.6 项目资源管理
**核心：** 确保资源可用并高效使用

**关注点：**
- 团队建设与发展
- 资源获取与分配
- 冲突管理
- 资源控制

#### 2.1.7 项目沟通管理
**核心：** 确保及时准确的信息交换

**关注点：**
- 沟通规划
- 信息分发
- 沟通效果监控
- 干系人反馈

#### 2.1.8 项目风险管理
**核心：** 最大化正面风险、最小化负面风险

**关注点：**
- 风险识别
- 风险分析（定性/定量）
- 风险应对策略
- 风险监控

#### 2.1.9 项目采购管理
**核心：** 确保外部资源获取

**关注点：**
- 采购规划
- 供方选择
- 合同管理
- 采购收尾

#### 2.1.10 项目干系人管理
**核心：** 最大化干系人支持

**关注点：**
- 干系人识别
- 干系人分析
- 参与策略
- 关系维护

### 2.2 项目整合管理 (Integration Management)

| 过程 | 说明 |
|------|------|
| 制定项目章程 | 编写一份正式批准项目的文件 |
| 制定项目管理计划 | 定义、准备和协调所有子计划 |
| 指导与管理项目工作 | 领导团队实现项目目标 |
| 管理项目知识 | 使用现有知识生成新知识 |
| 监控项目工作 | 跟踪、审查、报告整体进展 |
| 实施整体变更控制 | 审查、批准、拒绝变更请求 |
| 结束项目或阶段 | 终结项目或阶段的所有活动 |

**输入输出速查：**

| 过程        | 主要输入                 | 主要输出         |
| --------- | -------------------- | ------------ |
| 制定项目章程    | 商业文件、协议、事业环境因素       | 项目章程         |
| 制定项目管理计划  | 项目章程、事业环境因素          | 项目管理计划       |
| 指导与管理项目工作 | 项目管理计划、项目文件          | 可交付成果、工作绩效数据 |
| 管理项目知识    | 项目管理计划、项目文件          | 经验教训登记册      |
| 监控项目工作    | 项目管理计划、项目文件、工作绩效信息   | 变更请求、工作绩效报告  |
| 实施整体变更控制  | 项目管理计划、项目文件、变更请求     | 批准的变更请求、更新   |
| 结束项目或阶段   | 项目章程、项目管理计划、验收的可交付成果 | 最终报告、经验教训更新  |

## 七、核心工具与技术详解

### 7.1 挣值分析 (EVM)

| 指标 | 公式 | 含义 |
|------|------|------|
| PV | 计划价值 | 截止某时刻应该完成工作的预算 |
| EV | 挣值 | 截止某时刻实际完成工作的预算 |
| AC | 实际成本 | 截止某时刻实际花费 |
| SV | EV - PV | >0 进度超前，<0 进度落后 |
| CV | EV - AC | >0 成本节约，<0 成本超支 |
| SPI | EV / PV | >1 进度超前，<1 进度落后 |
| CPI | EV / AC | >1 成本节约，<1 成本超支 |
| EAC | 多种公式 | 完工估算 |
| VAC | BAC - EAC | 完工偏差 |

### 7.2 关键路径法 (CPM)

| 参数 | 说明 |
|------|------|
| ES | 最早开始时间 |
| EF | 最早完成时间 = ES + 工期 |
| LS | 最迟开始时间 = LF - 工期 |
| LF | 最迟完成时间 |
| TF | 总浮动时间 = LS - ES |
| FF | 自由浮动时间 |

### 7.3 风险矩阵

| 概率×影响 | 低影响 | 中影响 | 高影响 |
