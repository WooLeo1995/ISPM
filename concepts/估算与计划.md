# 估算与计划

## 估算类型

| 类型 | 何时做 | 准确性 |
|------|--------|--------|
| **类比估算** | 项目早期 | 低 (±40%) |
| **参数估算** | 有历史数据 | 中 (±15%) |
| **三点估算** | 需要考虑风险 | 中高 |
| **自下而上估算** | 详细规划 | 高 (±10%) |
| **专家判断** | 无数据参考 | 依赖专家 |

## 三点估算 (PERT)

### 公式

```
PERT期望 = (乐观 + 4×最可能 + 悲观) / 6
```

### 标准差

```
σ = (悲观 - 乐观) / 6
```

### 应用

- 考虑不确定性
- 概率分析
- 风险评估

## 进度计划方法

### 关键路径法 (CPM)

- 确定最长路径
- 计算浮动时间
- 识别关键活动

### 关键链法 (CCM)

- 考虑资源约束
- 添加项目缓冲
- 消除学生综合症

### 里程碑计划

```
□─────────□─────────□─────────□
M1        M2        M3        M4
```

## 资源优化

### 资源平衡 (Resource Leveling)

- 解决资源冲突
- 可能延长工期
- 在可用资源内优化

### 资源平滑 (Resource Smoothing)

- 不超过浮动时间
- 保持资源恒定
- 不延长工期

## 进度压缩

### 赶工 (Crashing)

| 特点 | 说明 |
|------|------|
| 方法 | 增加资源 |
| 成本 | 增加直接成本 |
| 限制 | 最短工期约束 |

### 快速跟进 (Fast Tracking)

| 特点 | 说明 |
|------|------|
| 方法 | 并行执行 |
| 成本 | 增加风险 |
| 限制 | 逻辑关系允许 |

## 成本估算

### 估算方法

| 方法 | 说明 |
|------|------|
| **类比估算** | 基于类似项目 |
| **自下而上** | 详细分解后汇总 |
| **参数模型** | 基于变量计算 |
| **供应商投标** | 基于市场报价 |

### 成本类型

| 类型 | 说明 |
|------|------|
| **直接成本** | 可直接归因 (人工、材料) |
| **间接成本** | 分摊成本 (水电、管理) |
| **固定成本** | 不随产量变化 |
| **可变成本** | 随产量变化 |

## 基准计划

```
范围基准 = 项目范围说明书 + WBS + WBS词典
进度基准 = 进度计划 + 里程碑
成本基准 = 成本预算
综合基准 = 整合所有子基准
```
