# 组织通用治理与管理

## 治理层次

| 层次 | 主体 | 关注点 |
|------|------|--------|
| **董事会** | 股东会、董事会 | 战略、合规 |
| **高管层** | CEO、高管 | 战略执行 |
| **管理层** | 中层 | 运营管理 |
| **执行层** | 基层 | 具体执行 |

## 公司治理

### 基本原则

1. 战略导向
2. 透明公开
3. 权责明确
4. 制衡机制
5. 持续改进

### 治理结构

| 机构 | 职责 |
|------|------|
| **股东会** | 最高权力机构 |
| **董事会** | 决策机构 |
| **监事会** | 监督机构 |
| **管理层** | 执行机构 |

## 战略管理

### 战略规划流程

```
战略分析 → 战略制定 → 战略选择 → 战略实施 → 战略控制
```

### 战略分析工具

| 工具 | 用途 |
|------|------|
| **PEST** | 宏观环境分析 |
| **波特五力** | 行业竞争分析 |
| **SWOT** | 内部外部分析 |
| **价值链分析** | 竞争优势 |

### PEST 分析

| 因素 | 内容 |
|------|------|
| **P 政治** | 法规、政策 |
| **E 经济** | 宏观经济 |
| **S 社会** | 人口、文化 |
| **T 技术** | 技术发展 |
