# 挣值管理 (Earned Value Management)

挣值管理是一种综合考虑**范围、进度、成本**的项目绩效测量方法。

## 三个核心指标

| 指标 | 英文 | 说明 |
|------|------|------|
| **计划价值** | PV (Planned Value) | 截至某时刻应该完成工作的预算价值 |
| **挣值** | EV (Earned Value) | 截至某时刻实际完成工作的预算价值 |
| **实际成本** | AC (Actual Cost) | 截至某时刻实际花费的成本 |

## 关键偏差指标

| 指标 | 公式 | 含义 |
|------|------|------|
| **进度偏差 (SV)** | SV = EV - PV | >0 进度超前，<0 进度落后 |
| **成本偏差 (CV)** | CV = EV - AC | >0 成本节约，<0 成本超支 |
| **进度绩效指数 (SPI)** | SPI = EV / PV | >1 进度超前，<1 进度落后 |
| **成本绩效指数 (CPI)** | CPI = EV / AC | >1 成本节约，<1 成本超支 |

## 完工预测

| 指标 | 公式 | 说明 |
|------|------|------|
| **完工预算 (BAC)** | - | 项目总预算 |
| **完工估算 (EAC)** | 多种计算方式 | 预测项目完工成本 |
| **完工偏差 (VAC)** | VAC = BAC - EAC | 预测成本与预算的差异 |
| **完工尚需估算 (ETC)** | ETC = EAC - AC | 预计还需多少成本 |

## 常用 EAC 公式

- `EAC = BAC / CPI` — 基于当前效率
- `EAC = AC + (BAC - EV)` — 基于原始计划
- `EAC = AC + ETC` — 重新估算

## 判断标准

| 状态 | SPI | CPI |
|------|-----|-----|
| 良好 | >0.9 | >0.9 |
| 关注 | 0.9~1.1 | 0.9~1.1 |
| 警告 | <0.9 | <0.9 |
