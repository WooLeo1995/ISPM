# 资源管理与团队建设

## 资源管理过程

### 1. 规划资源管理
- 制定资源管理计划
- 识别项目岗位与职责

### 2. 估算活动资源
- 确定所需资源类型和数量
- 输出：资源分解结构 (RBS)

### 3. 获取资源
- 分配资源到项目
- 解决资源冲突

### 4. 建设团队
- 培养团队能力
- 提高团队协作

### 5. 管理团队
- 跟踪团队绩效
- 解决冲突

### 6. 控制资源
- 确保资源可用
- 调整资源分配

## 资源分解结构 (RBS)

```
资源分解结构
├── 人员

│   ├── 技术人员
│   └── ...
├── 设备
├── 材料
└── 消耗品
```

## 团队发展阶段 (Tuckman)

| 阶段 | 特征 | 领导风格 |
|------|------|----------|
| **形成期** | 陌生、谨慎 | 指导型 |
| **震荡期** | 冲突、抵触 | 教练型 |
| **规范期** | 合作、建立规范 | 参与型 |
| **成熟期** | 高效、自组织 | 授权型 |
| **解散期** | 结束、总结 | - |

## 冲突管理

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **撤退/回避** | 推迟或退出 | 非紧急、情绪化 |
| **缓和/包容** | 强调一致 | 需维护关系 |
| **妥协/调解** | 各让一步 | 临时解决 |
| **强制/命令** | 强制执行 | 紧急决策 |
| **合作/解决问题** | 寻求共识 | 重要问题 |

> 合作/解决问题是最有效的冲突解决方法

|------|----------|
