# 合同类型与招标流程

## 合同法基础

### 合同订立

| 要素 | 说明 |
|------|------|
| 要约 | 向对方提出条件 |
| 承诺 | 接受要约 |
| 成立 | 承诺到达时生效 |

### 合同效力

| 类型 | 效力 |
|------|------|
| 有效合同 | 产生法律效力 |
| 无效合同 | 自始无效 |
| 可撤销 | 撤销前有效 |
| 效力待定 | 需追认 |

### 合同解除

| 方式 | 说明 |
|------|------|
| 协议解除 | 双方协商 |
| 法定解除 | 法定事由 |
| 约定解除 | 约定条件成就 |

## 工程合同分类

### 按计价方式

| 类型 | 特点 | 风险 |
|------|------|------|
| **总价合同** | 固定总价 | 卖方承担 |
| **单价合同** | 固定单价 | 双方共担 |
| **成本补偿合同** | 实报实销+利润 | 买方承担 |

### 合同选择矩阵

| 项目情况 | 推荐合同 |
|----------|----------|
| 范围明确 | 总价合同 |
| 技术复杂 | 成本补偿 |
| 紧急项目 | 成本补偿 |
| 长期项目 | 价格调整 |

## 招标流程

### 招标准备

1. 组建招标机构
2. 编制招标文件
3. 确定招标方式
4. 审查投标资质

### 招标阶段

| 步骤 | 活动 |
|------|------|
| 发布招标公告 | 公开招标 |
| 投标资格预审 | 筛选投标人 |
| 发售招标文件 | 提供技术规格 |
| 踏勘现场 | 了解项目情况 |
| 澄清答疑 | 解答疑问 |

### 投标阶段

1. 投标报名
2. 购买招标文件
3. 编制投标文件
4. 缴纳投标保证金
5. 递交投标文件

### 开标评标

| 环节 | 内容 |
|------|------|
| 开标 | 公开宣读报价 |
| 评标 | 评审投标文件 |
| 中标 | 确定中标人 |

### 评标方法

| 方法 | 说明 |
|------|------|
| **综合评估法** | 综合评分 |
| **最低价法** | 价格最低者 |
| **经评审的最低价** | 通过评审的最低价 |

## 政府采购

### 采购方式

| 方式 | 适用条件 |
|------|----------|
| 公开招标 | 主要方式 |
| 邀请招标 | 特殊项目 |
| 竞争性谈判 | 竞争不充分 |
| 单一来源 | 唯一供应商 |
| 询价 | 标准化产品 |

### 采购流程

```
需求确定 → 预算审批 → 方式选择 → 采购执行 → 验收付款
```
