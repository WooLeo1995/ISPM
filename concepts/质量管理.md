# 质量管理

## 质量管理的三个过程

### 1. 规划质量管理
- 识别项目质量标准
- 制定质量管理计划
- 输出：质量管理计划、质量指标

### 2. 实施质量保证 (QA)
- 按照质量管理计划执行
- 过程审计和改进
- 持续改进

### 3. 质量控制 (QC)
- 监控具体成果
- 识别质量问题
- 分析原因并纠治

## 核心工具与技术

| 工具 | 用途 |
|------|------|
| **质量审计** | 检查质量活动是否符合标准 |
| **过程分析** | 识别改进机会 |
| **七种基本质量工具** | 解决质量问题 |
| **测试/检查** | 验证成果质量 |

## 七种基本质量工具

| 工具 | 用途 |
|------|------|
| **因果图（鱼骨图）** | 分析问题根本原因 |
| **流程图** | 展示过程步骤 |
| **核查表** | 收集数据 |
| **帕累托图** | 识别主要问题（80/20法则） |
| **直方图** | 显示数据分布 |
| **控制图** | 监控过程稳定性 |
| **散点图** | 展示两变量关系 |

## 质量成本

| 成本类型 | 说明 |
|----------|------|
| **预防成本** | 培训、流程改进 |
| **评估成本** | 测试、检查 |
| **内部失败成本** | 内部发现缺陷的处理 |
| **外部失败成本** | 客户发现缺陷的处理 |

> 最佳实践：增加预防成本，减少失败成本

## 成熟度模型

| 级别 | 名称 | 特征 |
|------|------|------|
| 1 | 初始级 | 随机过程 |
| 2 | 可管理级 | 制度化 |
| 3 | 已定义级 | 标准化 |
| 4 | 量化管理级 | 度量化 |
| 5 | 持续优化级 | 持续改进 |
