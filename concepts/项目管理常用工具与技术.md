# 项目管理常用工具与技术

## 规划工具

### 思维导图

- 头脑风暴可视化
- 分解结构展示
- 关联关系呈现

### 网络图

- 活动节点图 (AON)
- 活动箭头图 (AOA)
- 展示活动逻辑关系

### 横道图 (Gantt)

```
活动 A  ████████
活动 B      ████████
活动 C          ████████
     Week1   Week2   Week3
```

## 分析工具

### SWOT 分析

|          | 正面 | 负面 |
|----------|------|------|
| **内部** | 优势 (Strengths) | 劣势 (Weaknesses) |
| **外部** | 机会 (Opportunities) | 威胁 (Threats) |

### 回归分析

- 确定变量关系
- 预测趋势
- 因果分析

### 趋势分析

- 基于历史数据
- 预测未来走向
- 偏差分析基础

## 决策工具

### 决策树

```
        [决策点]
       /    |    \
    方案A  方案B  方案C
     |      |      |
    收益   收益   收益
   / \    / \    / \
  好 差  好 差  好  差
```

### 多标准决策分析

| 准则 | 权重 | 方案A | 方案B |
|------|------|-------|-------|
| 成本 | 30% | 8 | 7 |
| 质量 | 40% | 9 | 8 |
| 风险 | 30% | 7 | 9 |

## 沟通工具

### 报告类型

| 类型 | 受众 | 频率 |
|------|------|------|
| 状态报告 | 项目团队 | 每周 |
| 进展报告 | 管理层 | 每月 |
| 预测报告 | 干系人 | 按需 |

### 会议纪要要素

1. 会议基本信息
2. 讨论内容
3. 决策事项
4. 行动项 (负责人 + 截止)
5. 下次会议安排
