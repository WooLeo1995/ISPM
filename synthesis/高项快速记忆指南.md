### 1️⃣ 整合管理（必考）

#### 七大过程流程图

```
启动阶段
  ↓
1. 制定项目章程 ← 发起人/高层批准
  ↓
规划阶段
  ↓
2. 制定项目管理计划 ← 整合所有子计划
  ↓
执行阶段
  ↓
3. 指导与管理项目工作 ← 执行计划，产生可交付成果
  ↓
4. 管理项目知识 ← 显性/隐性知识
  ↓
监控阶段
  ↓
5. 监控项目工作 ← 跟踪、审查、报告
  ↓
6. 实施整体变更控制 ← CCB 审批
  ↓
收尾阶段
  ↓
7. 结束项目或阶段 ← 验收、总结
```

#### 关键输出物

| 过程 | 输出 | 记忆点 |
|------|------|--------|
| 制定章程 | 项目章程 | 正式启动，授权 PM |
| 制定计划 | 项目管理计划 | 整合所有子计划 |
| 指导执行 | 可交付成果 | 项目产品 |
| 管理知识 | 经验教训登记册 | 组织资产 |
| 监控工作 | 变更请求、工作绩效报告 | 发现问题 |
| 变更控制 | 批准的变更请求 | CCB 决策 |
| 结束项目 | 最终产品、经验教训 | 项目完成 |

#### 变更控制流程（★★★ 必考）

```
变更请求提出
    ↓
变更影响分析（评估对范围、进度、成本、质量的影响）
    ↓
CCB 审批（变更控制委员会）
    ↓
    ├─ 批准 → 实施变更 → 验证 → 更新基准
    └─ 拒绝 → 通知相关方
```

**CCB 人员组成**：项目经理、变更管理负责人、配置管理员、变更顾问委员会

#### 六大过程

| 过程 | 输入 | 输出 | 核心工具 |
|------|------|------|----------|
| 规划范围管理 | 项目章程 | 范围管理计划 | 专家判断 |
| 收集需求 | 项目章程 | 需求文件、需求跟踪矩阵 | 访谈、焦点小组 |
| 定义范围 | 需求文件 | 项目范围说明书 | 产品分析 |
| 创建 WBS | 范围说明书 | 范围基准（WBS+词典） | 分解 |
| 确认范围 | 可交付成果 | 验收的可交付成果 | 检查、群体决策 |
| 控制范围 | 工作绩效信息 | 工作绩效报告 | 偏差分析 |

#### WBS 分解原则（★★★）

- **100% 原则**：包含所有工作，不遗漏
- **80 小时原则**：工作包不超过 80 小时
- **相互独立**：避免重复
- **滚动规划**：近期详细，远期粗略

#### 范围基准三要素

```
范围基准 = 项目范围说明书 + WBS + WBS 词典
```

### 3️⃣ 进度管理

#### 关键路径法（CPM）- 必考计算题

**六个时间参数**

| 参数 | 英文 | 说明 | 计算 |
|------|------|------|------|
| 最早开始 | ES | 活动最早能开始 | max(前导 EF) |
| 最早完成 | EF | 活动最早能完成 | ES + 工期 |
| 最迟开始 | LS | 活动最迟必须开始 | LF - 工期 |
| 最迟完成 | LF | 活动最迟必须完成 | min(后继 LS) |
| 总浮动时间 | TF | 可延迟时间 | LS - ES |
| 自由浮动时间 | FF | 不影响后续活动 | 后继 ES - 本活动 EF |

**关键路径特征**：TF = 0，任何延迟都影响项目

#### 进度压缩方法

| 方法 | 特点 | 成本 | 风险 |
|------|------|------|------|
| 赶工（Crashing） | 增加资源 | ↑ 增加 | ↑ 增加 |
| 快速跟进（Fast Tracking） | 并行执行 | → 不变 | ↑↑ 大幅增加 |

### 4️⃣ 成本管理 - 挣值管理（EVM）

#### 三个基本指标

| 指标 | 英文 | 说明 | 计算 |
|------|------|------|------|
| 计划价值 | PV | 应该完成工作的预算 | 计划工作量 × 单价 |
| 挣值 | EV | 实际完成工作的预算 | 实际完成工作量 × 单价 |
| 实际成本 | AC | 实际花费 | 实际支出 |

#### 偏差分析（★★★ 必考）

| 指标 | 公式 | 判断 | 记忆 |
|------|------|------|------|
| 进度偏差 | SV = EV - PV | >0 超前，<0 落后 | EV 减 PV |
| 成本偏差 | CV = EV - AC | >0 节约，<0 超支 | EV 减 AC |
| 进度绩效 | SPI = EV/PV | >1 超前，<1 落后 | EV 除 PV |
| 成本绩效 | CPI = EV/AC | >1 节约，<1 超支 | EV 除 AC |

**快速判断**：
- SPI < 0.9 → 进度严重落后
- CPI < 0.9 → 成本严重超支

#### 完工预测（EAC）

| 公式 | 适用场景 | 说明 |
|------|----------|------|
| EAC = BAC / CPI | 基于当前效率 | 假设未来效率不变 |
| EAC = AC + (BAC - EV) | 基于原始计划 | 假设未来按计划进行 |
| EAC = AC + ETC | 重新估算 | 基于新的估算 |

**完工偏差**：VAC = BAC - EAC

### 5️⃣ 质量管理

#### 三大过程

| 过程 | 输入 | 输出 | 工具 |
|------|------|------|------|
| 规划质量管理 | 项目管理计划 | 质量管理计划 | 成本效益分析 |
| 实施质量保证（QA） | 工作绩效数据 | 工作绩效信息 | 过程审计 |
| 质量控制（QC） | 可交付成果 | 验收的可交付成果 | 检查、测试 |

**QA vs QC 区分**
- QA：过程审计，预防问题
- QC：结果检查，发现问题

#### 七种基本质量工具

| 工具 | 用途 | 记忆 |
|------|------|------|
| 因果图（鱼骨图） | 分析根本原因 | 找原因 |
| 流程图 | 展示过程步骤 | 看流程 |
| 核查表 | 收集数据 | 打勾 |
| 帕累托图 | 80/20 法则 | 找主要问题 |
| 直方图 | 数据分布 | 看分布 |
| 控制图 | 监控稳定性 | 看趋势 |
| 散点图 | 两变量关系 | 看关系 |

#### 质量成本（COPQ）

| 类型 | 说明 | 趋势 |
|------|------|------|
| 预防成本 | 培训、流程改进 | ↑ 增加 |
| 评估成本 | 测试、检查 | ↑ 增加 |
| 内部失败成本 | 内部发现缺陷 | ↓ 减少 |
| 外部失败成本 | 客户发现缺陷 | ↓↓ 大幅减少 |

**黄金法则**：预防成本 > 失败成本，COPQ 越低越好

### 6️⃣ 风险管理

#### 七大过程

```
1. 规划风险管理 → 输出风险管理计划
2. 识别风险 → 输出风险登记册
3. 定性风险分析 → 更新风险登记册
4. 定量风险分析 → 更新风险登记册
5. 规划风险应对 → 输出变更请求
6. 实施风险应对 → 执行应对计划
7. 监督风险 → 监控风险状态
```

#### 风险应对策略（★★★）

| 策略 | 威胁应对 | 机会应对 | 记忆 |
|------|----------|----------|------|
| 上报 | 上报给更高层 | 上报给更高层 | 超出权限 |
| 规避 | 消除威胁 | 消除机会 | 不做 |
| 转移 | 保险、外包 | 伙伴关系 | 转给别人 |
| 减轻 | 减少概率/影响 | 增加概率/影响 | 降低程度 |
| 接受 | 应急储备 | 应急储备 | 承受 |

**记忆口诀**：上规转减接（上报、规避、转移、减轻、接受）

## 📊 计算题专题

### 1. 挣值管理计算

```
已知：PV、EV、AC、BAC

求：SV、CV、SPI、CPI、EAC、VAC

SV = EV - PV
CV = EV - AC
SPI = EV / PV
CPI = EV / AC
EAC = BAC / CPI
VAC = BAC - EAC
```

### 2. 关键路径计算

```
已知：活动、工期、前导活动

求：ES、EF、LS、LF、TF、FF

ES = max(前导 EF)
EF = ES + 工期
LF = min(后继 LS)
LS = LF - 工期
TF = LS - ES
FF = 后继 ES - 本活动 EF
```

### 3. 决策树分析

```
EMV = Σ(结果值 × 概率)
选择 EMV 最高的方案
```

### 4. 现值计算

```
PV = FV / (1 + r)^n
NPV = Σ(PV) - 初始投资
```

## 💡 记忆技巧

### 1. 联想记忆法

- **整合管理**：项目的"大脑"，协调所有工作
- **范围管理**：项目的"边界"，定义做什么
- **进度管理**：项目的"时间"，定义何时完成
- **成本管理**：项目的"金钱"，定义花多少

| 对比 | 区分 |
|------|------|
| QA vs QC | 预防 vs 检查 |
| 风险 vs 问题 | 未来 vs 现在 |
| 总价 vs 成本补偿 | 低风险 vs 高风险 |
